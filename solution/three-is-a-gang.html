Solution: Three is a Gang
Answer: EXECUTION

Written by Rio Futaba

This is a meta-matching puzzle. There are 27 members and 9 gangs, so we probably want to split the members into 9 gangs of 3 (this is also hinted by the title). Each gang represents a meta and we should find 3 sets of feeders that go into it. Here are the solutions to all the metas (sorted in alphabetical order of the gangs):

  • Commoners

Members:
{ANCIENT GREEK, DEBUT SINGLE, INSANE CLOWN}
{BOXING EXPERTS, ENGAGE IN, FIRST CLASS}
{COMMUNIST PARTY, GOTHIC REVIVAL, HOW LONG}

Possible break-in point: These are all two word phrases whose first letters span from A-I.

Sorting these answers in alphabetical order, we get

ANCIENT GREEK
BOXING EXPERTS
COMMUNIST PARTY
DEBUT SINGLE
ENGAGE IN
FIRST CLASS
GOTHIC REVIVAL
HOW LONG
INSANE CLOWN

The title Commoner clues that each pair of words have a common letter. Taking the common letter of each phrase (e.g. X from BOXING EXPERTS) and reading them in order gives EXTENSION.

  • Fortune Tellers

Members:
{DECISION THEORY, SOLITARY WAVE}
{DEMON HUNTERS}
{RESTRAINT ORDER}

Possible break-in point: The first words clue major arcana.

The title Fortune Tellers is a hint that the major arcana is related to this meta. Indeed, the first words of these answers all clue a major arcana:

RESTRAINT = Temperance (14)
DECISION = Judgement (20)
SOLITARY = The Hermit (9)
DEMON = The Devil (15)

Noting that the second words have lengths 4-7, we can reorder these answers by the length of the second word and extract using A1Z26 on the Major Arcanas.

SOLITARY WAVE = The Hermit (9) -> I
RESTRAINT ORDER = Temperance (14) -> N
DECISION THEORY = Judgement (20) -> T
DEMON HUNTERS = The Devil (15) -> O

This gives the answer INTO.

  • Loners

Members:
{CATALLACTIC, PARATRIPTIC, PINGING, STACCATOS}
{REVIVER, SAURURUS, UNENDURED, VOLVOCACEAE}
{PEEPED, TITANIA, UNSYNONYMOUS}

Possible break-in point: These answers have exactly one letter that appears exactly once.

Each of these answers have exactly one letter that appears exactly once (so they are the “loners”). Also, note that the positions of these letters are from 1 to 11, which gives us an ordering:

  • Mixed Numbers

Members:

{FORGIVENESS, HEIGHTENED, TETHERS}
{HIGH TECH, TAKEN OVER}

Possible break-in point: This set of answers have lengths 7-13. They also all contain anagrammed numbers.

The title Mixed Numbers hints that we should be looking for anagrams of numbers. Each of these answers contains an anagram of some number. We sort the answers by length and convert each number to a letter using A1Z26.

  • Oxymorons

Members:
{HOLY FIND, OBJECTIVE BASES}
{LIGHT BALLAT, UNPLANNED LOT}
{TASTELESS SPIES, TINY PLANT}

Possible break-in point: These phrases look like oxymorons with one letter missing.

We can add a letter to the second words of these phrases such that they become oxymorons. The positions of the missing letters range from 1-6, which gives us an ordering.

  • Pool of Idioms

Members:
{FREE PROSE}
{FOOD JOURNALISM}
{NICE PENCIL, WILL HERRING}

Possible break-in point: This one is kind of hard to get from the beginning. The only obvious common point is that the first words all have 4 letters.

These are all idioms where the first word, which is a color, is replaced by a 4-letter word. The colors are yellow, blue, red, and purple, which corresponds to the colors of balls 1-4 in pool (which is why the gang name is Pool of Idioms). Sorting by ball number and indexing the ball number into the first word gives the answer FILE.

  • Social Distancers

Members:
{CAVALRY DIVISION, FREE DOWNLOAD}
{DEFEATED, DIPLOMATIC}
{MARRIED COUPLE, SOCIAL STUDIES}

Possible break-in point: This one is almost impossible to get at the beginning.

Each of these feeders contain a letter which appears twice and there are exactly 6 letters between its two occurrences. Moreover, the positions of the first occurrence of these letters range from 1 to 6, which gives an ordering. Reading these letters in order gives the answer DIVIDE.

  • Stutterers

{FOOD SECURITY, HAPPY RETREAT}
{PRETTY RIDICULOUS, SEE BELOW}

Possible break-in point: The first words have lengths 3-8, and they all contain double letters.

The first word of these feeders all contain double letters. Furthermore, their lengths are 3-8, which gives an ordering. Taking the letters at the corresponding positions in the second word and reading in order gives the answer ELECTRICALLY.

  • Unordered Bigrams

Members:
{≤}
{I SEE}
{PASCAL, RETWEET}

Possible break-in point: Each of these feeders clue a bigram.

Each of these feeders clue a bigram:

≤LE (≤ is \le in LaTeX)
I SEEIC
PASCALPA (unit of pressure)
RETWEETRT

The gang name is Unordered Bigrams, so we should reorder the bigrams to form the answer PARTICLE.

Now, we have a list of 9 words from the meta answers:

  • EXTENSION
  • INTO
  • PALINDROMIC
  • CHARGED
  • PIECES
  • FILE
  • DIVIDE
  • ELECTRICALLY
  • PARTICLE

What should we do with these answers? The observation is that we can divide these words into three groups of three to form three crossword clues! We have

PALINDROMIC FILE EXTENSION -> EXE
DIVIDE INTO PIECES -> CUT
ELECTRICALLY CHARGED PARTICLE -> ION

Putting these three trigrams together gives us the final answer EXECUTION.

Author’s Notes

This puzzle was born after I realized that EXECUTION can be split into three trigrams, each of which can be the answer to a crossword clue.
