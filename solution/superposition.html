We spell out the message “OVERLAP GALLIFREYAN”. Gallifreyan is a language that appears in Doctor Who. Calling in this message gives us a link to a PDF guide for writing Sherman’s Gallifreyan, as well as a link to a translator.

The next step is to translate each of our answers to Gallireyan using these tools. For example, AGED and LEFT look like this:

We are told that we should “overlap” these two. If we do this, we see that some curves and lines share the same positions:

This corresponds to a letter in Gallifreyan. They both share the base “B” shape and have a line connecting it, which translates to the letter “G”. There is no other position where the two diagrams share a curve or line.

Note that when using the translator, the lines won’t always stay in the same position. This is fine, because when determining whether lines coincide between the two diagrams, what matters is the number of lines that connect to the base shape.
