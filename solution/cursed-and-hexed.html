Solution: Cursed and Hexed
Answer: GRAVY BOAT

Written by lightbulbMEOW

The first step is to notice that each file is encrypted in some way, and the clues themselves are also encoded in some way.

File Encrypted file with Encoded clues
1.avi Backwards English to Spanish
2.gif Base64 Caesar shift (A->B, 1->2)
3.html Caesar shift (A->B, 1->2) Swap adjacent letters
4.jpeg Each byte is double Inverted bits (00 -> FF)
5.mpeg Swap adjacent bits Interleave halves
6.pdf Interleaved with ABCDEF… Each letter is double
7.png Inverted bits (00 -> FF) Base64
8.svg English to Spanish Interleaved with ABCDEF…
9.wav Interleave halves Backwards

Notice that the file extensions are in alphabetical order, which suggests we must reorder them. The encryption/encoding methods form a cycle, which provide the ordering. The next step is to solve these subpuzzles. Each of these give either three or four words as the answer.

File Puzzle type Answer
PNG Various diagrams ELEMENTS MINESWEEPER GROUND
GIF Nikoli logic puzzles TREEHOUSE SUBDUE RESPONDS
HTML Crossword INDENT DEFAULT CLASSES ALIGN
MPEG Rickroll :) JOKESTER DADDY FOOL VIDEOS
WAV Various encodings YACHT LIGHT SURFRIDING
AVI Spanish clues SATURDAY MONTHS BASEBALL
SVG Various encodings SECRETLY INSIDERS ONLINE
PDF Math questions LIMIT MULTIPLICITY ELLIPSE APERY
JPEG Logos CHROMIUM TWITTER KELLOGGS

Notice that each of these answers matches the full name of the extension of the next file in the cycle. For example, solving the PNG file gives "ELEMENTS MINESWEEPER GROUND", which matches the enumeration of the full name of GIF, "Graphics Interchange Format".

To extract, take their common letters. The common letters spell out "SEE UNUSED FIRST LETTERS".

Common letters Unused word

Po(R)table Network Graphic(S) TWITTER

Taking the first letters of the unused words spell out GRAVY BOAT.
